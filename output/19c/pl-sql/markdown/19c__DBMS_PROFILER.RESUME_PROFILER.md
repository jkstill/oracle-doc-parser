# DBMS_PROFILER.RESUME_PROFILER

This function resumes profiler data collection.

## Syntax

```sql
DBMS_PROFILER.RESUME_PROFILER 
  RETURN BINARY_INTEGER; 

DBMS_PROFILER.RESUME_PROFILER;
```

**Returns:** `BINARY_INTEGER`

# DBMS_RESOURCE_MANAGER.BEGIN_SQL_BLOCK

This procedure, to be used with parallel statement queuing, indicates the start of a block of SQL statements that should be treated as a group by resource manager.

## Syntax

```sql
DBMS_RESOURCE_MANAGER.BEGIN_SQL_BLOCK;
```

## Usage Notes

Syntax DBMS_RESOURCE_MANAGER.BEGIN_SQL_BLOCK; Usage Notes For more information, see "Parallel Statement Queuing" and "Managing Parallel Statement Queuing with Resource Manager" in Oracle Database VLDB and Partitioning Guide .

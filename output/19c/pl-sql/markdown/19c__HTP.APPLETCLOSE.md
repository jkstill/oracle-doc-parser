# HTP.APPLETCLOSE

This procedure closes the applet invocation with the </APPLET> tag.

## Syntax

```sql
HTP.APPLETCLOSE;
```

## Usage Notes

You must first invoke the a Java applet using APPLETOPEN Procedure . Syntax HTP.APPLETCLOSE;

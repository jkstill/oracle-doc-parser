# HTF.WBR

This function generates the <WBR> tag, which inserts a soft line break within a section of NOBR text.

## Syntax

```sql
HTF.WBR
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

Syntax HTF.WBR RETURN VARCHAR2; Examples This function generates <WBR>

# HTP.BLOCKQUOTECLOSE

This procedure generates the </BLOCKQUOTE> tag which mark the end of a section of quoted text.

## Syntax

```sql
HTP.BLOCKQUOTECLOSE;
```

## Usage Notes

You mark the beginning of a section of text by means of the BLOCKQUOTEOPEN Procedure . Syntax HTP.BLOCKQUOTECLOSE; Examples This procedure generates </BLOCKQUOTE>

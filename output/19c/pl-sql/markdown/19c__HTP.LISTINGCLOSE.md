# HTP.LISTINGCLOSE

This procedure generates the </LISTING> tags which marks the end of a section of fixed-width text in the body of an HTML page.

## Syntax

```sql
HTP.LISTINGCLOSE;
```

## Usage Notes

To mark the beginning of a section of fixed-width text in the body of an HTML page, use the LISTINGOPEN Procedure . Syntax HTP.LISTINGCLOSE; Examples This procedure generates </LISTING>

# HTF.HTMLOPEN

This function generates the <HTML> tag which marks the beginning of an HTML document.

## Syntax

```sql
HTF.HTMLOPEN
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You use the HTMLCLOSE Function to mark the end of the an HTML document. Syntax HTF.HTMLOPEN RETURN VARCHAR2; Examples This function generates <HTML>

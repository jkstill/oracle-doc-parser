# DBMS_SQLDIAG

DBMS_SQLDIAG defines constants to use when specifying parameter values.

## Usage Notes

These constants are shown in the following tables: Table 165-1 describes the name of SQL repair advisor as seen by the advisor framework Table 165-2 describes SQLDIAG advisor task scope parameter values Table 165-3 describes SQLDIAG advisor time_limit constants Table 165-4 describes possible formats for a report Table 165-5 describes possible levels of detail in the report Table 165-6 describes possible report sections (comma delimited) Table 165-7 describes possible values for the problem_type parameter of the CREATE_DIAGNOSIS_TASK Functions Table 165-8 describes possible values for the _sql_findings_mode parameter

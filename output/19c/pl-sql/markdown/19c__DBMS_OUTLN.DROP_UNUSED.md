# DBMS_OUTLN.DROP_UNUSED

This procedure drops outlines that have never been applied in the compilation of a SQL statement.

## Syntax

```sql
DBMS_OUTLN.DROP_UNUSED;
```

## Usage Notes

Syntax DBMS_OUTLN.DROP_UNUSED; Usage Notes You can use DROP_UNUSED for outlines generated by an application for one-time use SQL statements created as a result of dynamic SQL. These outlines are never used and take up valuable disk space.

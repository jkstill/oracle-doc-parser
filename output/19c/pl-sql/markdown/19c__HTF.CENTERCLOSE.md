# HTF.CENTERCLOSE

This function generates the </CENTER> tag which marks the end of a section of text to center.

## Syntax

```sql
HTF.CENTERCLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You mark the beginning of a section of text to center with the CENTEROPEN Function . Syntax HTF.CENTERCLOSE RETURN VARCHAR2; Examples This function generates </CENTER>

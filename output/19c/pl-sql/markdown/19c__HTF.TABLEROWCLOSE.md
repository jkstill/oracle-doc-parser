# HTF.TABLEROWCLOSE

This function generates the </TR> tag which marks the end of a new row in an HTML table.

## Syntax

```sql
HTF.TABLEROWCLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

To mark the beginning of a new row, use the TABLEROWOPEN Function . Syntax HTF.TABLEROWCLOSE RETURN VARCHAR2; Examples This function generates </TABLE>

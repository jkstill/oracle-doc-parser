# HTP.FONTCLOSE

This procedure generates the </FONT> tag which marks the end of a section of text with the specified font characteristics.

## Syntax

```sql
HTP.FONTCLOSE;
```

## Usage Notes

You mark the beginning of the section text by means of the FONTOPEN Procedure . Syntax HTP.FONTCLOSE; Examples This procedure generates </FONT>

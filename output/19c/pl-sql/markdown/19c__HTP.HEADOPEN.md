# HTP.HEADOPEN

This procedure generates the <HEAD> tag which marks the beginning of the HTML document head section.

## Syntax

```sql
HTP.HEADOPEN;
```

## Usage Notes

You mark the end of an HTML document head section by means of the HEADCLOSE Procedure . Examples This procedure generates <HEAD>

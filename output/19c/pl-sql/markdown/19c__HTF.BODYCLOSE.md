# HTF.BODYCLOSE

This function generates the </BODY> tag which marks the end of a body section of an HTML document.

## Syntax

```sql
HTF.BODYCLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You mark the beginning of a body section by means of the BODYOPEN Function . Syntax HTF.BODYCLOSE RETURN VARCHAR2; Examples This function generates </BODY>

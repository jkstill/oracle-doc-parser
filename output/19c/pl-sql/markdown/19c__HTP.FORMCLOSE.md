# HTP.FORMCLOSE

This procedure generates the </FORM> tag which marks the end of a form section in an HTML document.

## Syntax

```sql
HTP.FORMCLOSE;
```

## Usage Notes

You mark the beginning of the form section by means of the FORMOPEN Procedure . Syntax HTP.FORMCLOSE; Examples This procedure generates </FORM>

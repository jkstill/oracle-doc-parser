# DBMS_XDB_CONFIG.GETHTTPCONFIGREALM

This function gets the realm name. Definition of a realm is referenced in IETF's RFC2617.

## Syntax

```sql
DBMS_XDB_CONFIG.GETHTTPCONFIGREALM
 RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

Syntax DBMS_XDB_CONFIG.GETHTTPCONFIGREALM RETURN VARCHAR2;

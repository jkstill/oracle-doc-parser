# HTP.HTMLCLOSE

This procedure generates the </HTML> tag which marks the end of an HTML document.

## Syntax

```sql
HTP.HTMLCLOSE;
```

## Usage Notes

You use the HTMLOPEN Procedure to mark the beginning of an HTML document. Syntax HTP.HTMLCLOSE; Examples This procedure generates </HTML>

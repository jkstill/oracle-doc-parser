# DBMS_XSTREAM_ADM.GET_TAG

This function gets the binary tag for all redo entries generated by the current session.

## Syntax

```sql
DBMS_XSTREAM_ADM.GET_TAG
RETURN RAW;
```

**Returns:** `RAW`

## Usage Notes

See Also: " SET_TAG Procedure " See Also: " SET_TAG Procedure " Syntax DBMS_XSTREAM_ADM.GET_TAG RETURN RAW; Examples The following example illustrates how to display the current logical change record (LCR) tag as output: SET SERVEROUTPUT ON DECLARE raw_tag RAW(2000); BEGIN raw_tag := DBMS_XSTREAM_ADM.GET_TAG(); DBMS_OUTPUT.PUT_LINE('Tag Value = ' || RAWTOHEX(raw_tag)); END; / You can also display the value by querying the DUAL view: SELECT DBMS_XSTREAM_ADM.GET_TAG FROM DUAL;

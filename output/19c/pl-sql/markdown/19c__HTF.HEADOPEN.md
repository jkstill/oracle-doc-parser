# HTF.HEADOPEN

This function generates the <HEAD> tag which marks the beginning of the HTML document head section.

## Syntax

```sql
HTF.HEADOPEN
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You mark the end of an HTML document head section by means of the HEADCLOSE Function . Syntax HTF.HEADOPEN RETURN VARCHAR2; Examples This function generates <HEAD>

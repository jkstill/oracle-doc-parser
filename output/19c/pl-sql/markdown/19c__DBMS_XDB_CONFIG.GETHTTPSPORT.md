# DBMS_XDB_CONFIG.GETHTTPSPORT

This procedure gets the value of the current HTTPS port.

## Syntax

```sql
DBMS_XDB_CONFIG.GETHTTPSPORT
 RETURN NUMBER;
```

**Returns:** `NUMBER`

## Usage Notes

Syntax DBMS_XDB_CONFIG.GETHTTPSPORT RETURN NUMBER; Usage Notes Returns NULL if no port has been configured

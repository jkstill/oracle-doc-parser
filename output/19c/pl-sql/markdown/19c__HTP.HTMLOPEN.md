# HTP.HTMLOPEN

This procedure generates the <HTML> tag which marks the beginning of an HTML document.

## Syntax

```sql
HTP.HTMLOPEN;
```

## Usage Notes

You use the HTMLCLOSE Procedure to mark the end of the an HTML document. Examples This procedure generates <HTML>

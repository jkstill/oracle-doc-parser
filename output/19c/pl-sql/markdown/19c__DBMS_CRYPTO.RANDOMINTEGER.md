# DBMS_CRYPTO.RANDOMINTEGER

This function returns an integer in the complete range available for the Oracle BINARY_INTEGER datatype.

## Syntax

```sql
DBMS_CRYPTO.RANDOMINTEGER
 RETURN BINARY_INTEGER;
```

**Returns:** `BINARY_INTEGER`

## Usage Notes

Syntax DBMS_CRYPTO.RANDOMINTEGER RETURN BINARY_INTEGER; Pragmas pragma restrict_references(randominteger,WNDS,RNDS,WNPS,RNPS);

# OWA_CUSTOM.AUTHORIZE

This function is used when PLSQL Agent's authorization scheme is set to GLOBAL or CUSTOM when there is no overriding OWA_CUSTOM package.

## Syntax

```sql
OWA_CUSTOM.AUTHORIZE 
  RETURN BOOLEAN;
```

**Returns:** `BOOLEAN`

## Usage Notes

Syntax OWA_CUSTOM.AUTHORIZE RETURN BOOLEAN;

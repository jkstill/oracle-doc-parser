# DBMS_DBFS_CONTENT.FLUSHSTATS

This procedure flushes DBMS_DBFS_CONTENT statistics to disk.

## Syntax

```sql
DBMS_DBFS_CONTENT.FLUSHSTATS;
```

**Returns:** `UNKNOWN`

## Usage Notes

# UTL_HTTP.GET_DETAILED_SQLCODE

This function retrieves the detailed SQLCODE of the last exception raised.

## Syntax

```sql
UTL_HTTP.GET_DETAILED_SQLCODE 
RETURN PLS_INTEGER;
```

**Returns:** `PLS_INTEGER`

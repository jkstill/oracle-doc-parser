# OWA_UTIL.GET_OWA_SERVICE_PATH

This function returns the full virtual path of the PL/SQL Gateway that is handling the request.

## Syntax

```sql
OWA_UTIL.GET_OWA_SERVICE_PATH
 RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

Syntax OWA_UTIL.GET_OWA_SERVICE_PATH RETURN VARCHAR2; Return Values A virtual path of the PL/SQL Gateway that is handling the request.

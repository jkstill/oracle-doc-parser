# HTP.FORMTEXTAREACLOSE

This procedure generates the </TEXTAREA> tag which ends a text area form element.

## Syntax

```sql
HTP.FORMTEXTAREACLOSE;
```

## Usage Notes

You open a text area element by means of either FORMTEXTAREAOPEN Procedure or FORMTEXTAREAOPEN2 Procedure . Syntax HTP.FORMTEXTAREACLOSE; Examples This procedure generates </TEXTAREA>

# HTF.FORMTEXTAREACLOSE

This function generates the </TEXTAREA> tag which ends a text area form element.

## Syntax

```sql
HTF.FORMTEXTAREACLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You open a text area element by means of either FORMTEXTAREAOPEN Function or FORMTEXTAREAOPEN2 Function . Syntax HTF.FORMTEXTAREACLOSE RETURN VARCHAR2; Examples This function generates </TEXTAREA>

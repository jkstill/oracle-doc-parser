# HTP.NOFRAMESOPEN

This procedure generates the <NOFRAMES> tag which mark the beginning of a no-frames section.

## Syntax

```sql
HTP.NOFRAMESOPEN;
```

## Usage Notes

To mark the end of a no-frames section, use the FRAMESETCLOSE Procedure . See also FRAME Procedure , FRAMESETOPEN Procedure and FRAMESETCLOSE Procedure . Syntax HTP.NOFRAMESOPEN; Examples This procedure generates <NOFRAMES>

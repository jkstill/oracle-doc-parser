# HTP.CENTERCLOSE

This procedure generates the </CENTER> tag which marks the end of a section of text to center.

## Syntax

```sql
HTP.CENTERCLOSE;
```

## Usage Notes

You mark the beginning of a section of text to center with the CENTEROPEN Procedure . Syntax HTP.CENTERCLOSE; Examples This procedure generates </CENTER>

# DBMS_XDB_CONFIG.CFG_GET

This function retrieves the session's configuration information as an XMLType instance.

## Syntax

```sql
DBMS_XDB_CONFIG.CFG_GET 
 RETURN SYS.XMLType;
```

**Returns:** `SYS.XMLType`

## Usage Notes

Syntax DBMS_XDB_CONFIG.CFG_GET RETURN SYS.XMLType;

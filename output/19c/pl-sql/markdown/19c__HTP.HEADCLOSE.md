# HTP.HEADCLOSE

This procedure generates the </HEAD> tag which marks the end of an HTML document head section.

## Syntax

```sql
HTP.HEADCLOSE;
```

## Usage Notes

You mark the beginning of an HTML document head section by means of the HEADOPEN Procedure . Syntax HTP.HEADCLOSE; Examples This procedure generates </HEAD>

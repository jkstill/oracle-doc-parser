# DBMS_AUDIT_UTIL

The DBMS_AUDIT_UTIL package provides functions that enable you to format the output of queries to the DBA_FGA_AUDIT_TRAIL , DBA_AUDIT_TRAIL , UNIFIED_AUDIT_TRAIL , and V$XML_AUDIT_TRAIL views.

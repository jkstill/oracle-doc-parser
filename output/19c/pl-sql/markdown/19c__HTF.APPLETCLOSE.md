# HTF.APPLETCLOSE

This function closes the applet invocation with the </APPLET> tag.

## Syntax

```sql
HTF.APPLETCLOSE
 RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You must first invoke the a Java applet using APPLETOPEN Function Syntax HTF.APPLETCLOSE RETURN VARCHAR2;

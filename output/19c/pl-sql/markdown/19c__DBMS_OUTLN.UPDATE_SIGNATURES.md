# DBMS_OUTLN.UPDATE_SIGNATURES

This procedure updates outline signatures to the current version's signature.

## Syntax

```sql
DBMS_OUTLN.UPDATE_SIGNATURES;
```

## Usage Notes

Syntax DBMS_OUTLN.UPDATE_SIGNATURES; Usage Notes You should execute this procedure if you have imported outlines generated in an earlier release to ensure that the signatures are compatible with the current release's computation algorithm.

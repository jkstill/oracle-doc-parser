# DBMS_RANDOM.NORMAL

This function returns random numbers in a standard normal distribution.

## Syntax

```sql
DBMS_RANDOM.NORMAL
  RETURN NUMBER;
```

## Parameters

| Parameter | Type | Mode | Description |
|-----------|------|------|-------------|
| number |  |  | Returns a random number |

**Returns:** `NUMBER`

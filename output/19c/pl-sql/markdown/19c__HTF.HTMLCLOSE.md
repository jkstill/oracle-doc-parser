# HTF.HTMLCLOSE

This function generates the </HTML> tag which marks the end of an HTML document.

## Syntax

```sql
HTF.HTMLCLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You use the HTMLOPEN Function to mark the beginning of an HTML document. Syntax HTF.HTMLCLOSE RETURN VARCHAR2; Examples This function generates </HTML>

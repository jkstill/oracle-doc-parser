# DBMS_XDBZ.PURGELDAPCACHE

This function purges the LDAP nickname cache.

## Syntax

```sql
DBMS_XDBZ.PURGELDAPCACHE
 RETURN BOOLEAN;
```

**Returns:** `BOOLEAN`

## Usage Notes

It returns TRUE if successful. Syntax DBMS_XDBZ.PURGELDAPCACHE RETURN BOOLEAN;

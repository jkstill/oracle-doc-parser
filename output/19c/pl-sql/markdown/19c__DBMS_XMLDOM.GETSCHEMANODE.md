# DBMS_XMLDOM.GETSCHEMANODE

This function retrieves the schema URI associated with the node.

## Syntax

```sql
DBMS_XMLDOM.GETSCHEMANODE(
   n       IN     DOMNODE)
 RETURN DOMNODE;
```

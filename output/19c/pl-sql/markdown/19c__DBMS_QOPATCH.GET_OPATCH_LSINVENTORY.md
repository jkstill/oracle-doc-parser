# DBMS_QOPATCH.GET_OPATCH_LSINVENTORY

This function returns whole opatch inventory as XML instance document.

## Syntax

```sql
DBMS_QOPATCH.GET_OPATCH_LSINVENTORY
 RETURN XMLTYPE;
```

**Returns:** `XMLTYPE`

## Usage Notes

Syntax DBMS_QOPATCH.GET_OPATCH_LSINVENTORY RETURN XMLTYPE;

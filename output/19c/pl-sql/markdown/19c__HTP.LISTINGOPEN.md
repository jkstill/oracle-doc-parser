# HTP.LISTINGOPEN

This procedure generates the <LISTING> tag which marks the beginning of a section of fixed-width text in the body of an HTML page.

## Syntax

```sql
HTP.LISTINGOPEN;
```

## Usage Notes

To mark the end of a section of fixed-width text in the body of an HTML page, use the LISTINGCLOSE Procedure . Syntax HTP.LISTINGOPEN; Examples This procedure generates <LISTING>

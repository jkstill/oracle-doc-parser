# HTP.MAPCLOSE

This procedure generates the </MAP> tag which marks the end of a set of regions in a client-side image map.

## Syntax

```sql
HTP.MAPCLOSE;
```

## Usage Notes

To mark the beginning of a set of regions in a client-side image map, use the MAPOPEN Procedure . Examples This procedure generates </MAP>

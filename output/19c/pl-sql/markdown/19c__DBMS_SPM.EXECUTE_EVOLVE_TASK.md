# DBMS_SPM.EXECUTE_EVOLVE_TASK

The function executes a previously created evolve task.

## Syntax

```sql
DBMS_SPM.EXECUTE_EVOLVE_TASK  (
   task_name        IN  VARCHAR2,
   execution_name   IN  VARCHAR2  := NULL,
   execution_desc   IN  VARCHAR2  := NULL);
 RETURN VARCHAR2;
```

## Parameters

| Parameter | Type | Mode | Description |
|-----------|------|------|-------------|
| task_name | VARCHAR2 | IN | Evolve task name |
| execution_name | VARCHAR2 | IN | Name to qualify and identify an execution. If not specified, it is generated by the advisor and returned by the function. |
| execution_desc | VARCHAR2 | IN | Description of the execution (maximum 256 characters) |

**Returns:** `VARCHAR2`

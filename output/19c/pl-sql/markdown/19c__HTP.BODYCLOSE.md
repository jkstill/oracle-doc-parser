# HTP.BODYCLOSE

This procedure generates the </BODY> tag which marks the end of a body section of an HTML document.

## Syntax

```sql
HTP.BODYCLOSE;
```

## Usage Notes

You mark the beginning of a body section by means of the BODYOPEN Procedure . Syntax HTP.BODYCLOSE; Examples This procedure generates </BODY>

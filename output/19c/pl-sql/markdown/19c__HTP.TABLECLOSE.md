# HTP.TABLECLOSE

This procedure generates the </TABLE> tag which marks the end of an HTML table.

## Syntax

```sql
HTP.TABLECLOSE;
```

## Usage Notes

To define the beginning of an HTML table, use the TABLEOPEN Procedure . Syntax HTP.TABLECLOSE; Examples This procedure generates </TABLE>

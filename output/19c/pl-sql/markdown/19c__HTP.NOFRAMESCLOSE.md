# HTP.NOFRAMESCLOSE

This procedure generates the </NOFRAMES> tag which marks the end of a no-frames section.

## Syntax

```sql
HTP.NOFRAMESCLOSE;
```

## Usage Notes

To mark the beginning of a no-frames section, use the FRAMESETOPEN Procedure . See also FRAME Procedure , FRAMESETOPEN Procedure and FRAMESETCLOSE Procedure . Syntax HTP.NOFRAMESCLOSE; Examples This procedure generates </NOFRAMES>

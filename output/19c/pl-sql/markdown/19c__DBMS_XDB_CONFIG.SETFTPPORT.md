# DBMS_XDB_CONFIG.SETFTPPORT

This procedure sets the FTP port to a new value.

## Syntax

```sql
DBMS_XDB_CONFIG.SETFTPPORT(
   new_port  IN  NUMBER);
```

## Parameters

| Parameter | Type | Mode | Description |
|-----------|------|------|-------------|
| new_port | NUMBER) | IN | Value to which the FTP port is set |

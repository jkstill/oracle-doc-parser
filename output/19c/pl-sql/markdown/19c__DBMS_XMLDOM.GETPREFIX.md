# DBMS_XMLDOM.GETPREFIX

This function retrieves the namespace prefix of the node.

## Syntax

```sql
DBMS_XMLDOM.GETPREFIX(
   n       IN     DOMNODE)
 RETURN VARCHAR2;
```

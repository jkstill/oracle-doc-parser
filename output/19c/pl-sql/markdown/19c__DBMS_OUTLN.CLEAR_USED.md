# DBMS_OUTLN.CLEAR_USED

This procedure clears the outline 'used' flag.

## Syntax

```sql
DBMS_OUTLN.CLEAR_USED (
   name    IN    VARCHAR2);
```

## Parameters

| Parameter | Type | Mode | Description |
|-----------|------|------|-------------|
| name | VARCHAR2) | IN | Name of the outline. |

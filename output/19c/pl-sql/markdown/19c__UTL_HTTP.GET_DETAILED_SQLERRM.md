# UTL_HTTP.GET_DETAILED_SQLERRM

This function retrieves the detailed SQLERRM of the last exception raised.

## Syntax

```sql
UTL_HTTP.GET_DETAILED_SQLERRM 
RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

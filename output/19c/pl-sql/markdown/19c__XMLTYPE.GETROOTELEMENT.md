# XMLTYPE.GETROOTELEMENT

this member function gets the root element of the XMLType instance. It returns NULL if the instance is a fragment.

## Syntax

```sql
MEMBER FUNCTION getRootElement
return varchar2 deterministic;
```

**Returns:** `varchar2`

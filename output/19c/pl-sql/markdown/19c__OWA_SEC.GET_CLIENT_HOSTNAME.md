# OWA_SEC.GET_CLIENT_HOSTNAME

This function returns the hostname of the client.

## Syntax

```sql
OWA_SEC.GET_CLIENT_HOSTNAME
 RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

Syntax OWA_SEC.GET_CLIENT_HOSTNAME RETURN VARCHAR2; Return Values The hostname.

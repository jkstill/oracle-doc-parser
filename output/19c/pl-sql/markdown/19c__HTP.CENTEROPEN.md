# HTP.CENTEROPEN

This procedure generates the <CENTER> tag which mark the beginning of a section of text to center.

## Syntax

```sql
HTP.CENTEROPEN;
```

## Usage Notes

You mark the beginning of a of a section of text to center by means of the CENTERCLOSE Procedure . Syntax HTP.CENTEROPEN; Examples This procedure generates <CENTER>

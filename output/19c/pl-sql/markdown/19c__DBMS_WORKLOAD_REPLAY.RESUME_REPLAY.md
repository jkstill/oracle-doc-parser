# DBMS_WORKLOAD_REPLAY.RESUME_REPLAY

This procedure resumes a paused workload replay.

## Syntax

```sql
DBMS_WORKLOAD_REPLAY.RESUME_REPLAY;
```

## Usage Notes

Syntax DBMS_WORKLOAD_REPLAY.RESUME_REPLAY; Usage Notes Prerequisite: A call to the PAUSE_REPLAY Procedure must have already been issued.

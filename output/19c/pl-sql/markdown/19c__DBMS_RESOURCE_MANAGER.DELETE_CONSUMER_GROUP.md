# DBMS_RESOURCE_MANAGER.DELETE_CONSUMER_GROUP

This procedure deletes entries which define resource consumer groups.

## Syntax

```sql
DBMS_RESOURCE_MANAGER.DELETE_CONSUMER_GROUP (
   consumer_group IN VARCHAR2);
```

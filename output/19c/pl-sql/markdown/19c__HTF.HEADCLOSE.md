# HTF.HEADCLOSE

This function generates the </HEAD> tag which marks the end of an HTML document head section.

## Syntax

```sql
HTF.HEADCLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You mark the beginning of an HTML document head section by means of the HEADOPEN Function . Syntax HTF.HEADCLOSE RETURN VARCHAR2; Examples This function generates </HEAD>

# DBMS_COMPARISON.DROP_COMPARISON

This procedure drops a comparison.

## Syntax

```sql
DBMS_COMPARISON.DROP_COMPARISON(
   comparison_name  IN  VARCHAR2);
```

## Parameters

| Parameter | Type | Mode | Description |
|-----------|------|------|-------------|
| comparison_name | VARCHAR2) | IN | The name of the comparison. |

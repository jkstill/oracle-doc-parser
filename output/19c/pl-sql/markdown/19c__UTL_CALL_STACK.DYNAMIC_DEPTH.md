# UTL_CALL_STACK.DYNAMIC_DEPTH

This function returns the number of subprograms on the call stack.

## Syntax

```sql
UTL_CALL_STACK.DYNAMIC_DEPTH
 RETURN PLS_INTEGER;
```

**Returns:** `PLS_INTEGER`

## Usage Notes

Syntax UTL_CALL_STACK.DYNAMIC_DEPTH RETURN PLS_INTEGER; Return Values The number of subprograms on the call stack

# DBMS_STORAGE_MAP.UNLOCK_MAP

This procedure unlocks the mapping information in the shared memory of the instance.

## Syntax

```sql
DBMS_STORAGE_MAP.UNLOCK_MAP;
```

## Usage Notes

Syntax DBMS_STORAGE_MAP.UNLOCK_MAP;

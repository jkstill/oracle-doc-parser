# DBMS_OUTLN.EXACT_TEXT_SIGNATURES

This procedure updates outline signatures to those that compute based on exact text matching.

## Syntax

```sql
DBMS_OUTLN.EXACT_TEXT_SIGNATURES;
```

## Usage Notes

Syntax DBMS_OUTLN.EXACT_TEXT_SIGNATURES; Usage Notes This procedure is relevant only for downgrading an outline to 8.1.6 or earlier.

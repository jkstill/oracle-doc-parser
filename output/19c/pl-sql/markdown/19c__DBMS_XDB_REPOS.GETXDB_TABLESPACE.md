# DBMS_XDB_REPOS.GETXDB_TABLESPACE

This function returns the current tablespace of the XDB (user).

## Syntax

```sql
DBMS_XDB_REPOS.GETXDB_TABLESPACE
 RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

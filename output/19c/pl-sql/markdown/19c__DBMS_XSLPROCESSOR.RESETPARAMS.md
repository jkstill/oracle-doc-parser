# DBMS_XSLPROCESSOR.RESETPARAMS

This procedure resets the top-level stylesheet parameters.

## Syntax

```sql
DBMS_XSLPROCESSOR.RESETPARAMS(
   ss  IN   Stylesheet);
```

|-----------|------|------|-------------|

# INTRODUCTION.REFERENCING

When you want to reference the types, items, and subprograms declared in a package specification, use the dot notation.

## Syntax

```sql
package_name.type_name
package_name.item_name
package_name.subprogram_name
```

## Usage Notes

For example: package_name.type_name package_name.item_name package_name.subprogram_name

# HTF.PARA

This function generates the <P> tag which indicates that the text that comes after the tag is to be formatted as a paragraph.

## Syntax

```sql
HTF.PARA
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You can add attributes to the tag by means of the PARAGRAPH Function . Syntax HTF.PARA RETURN VARCHAR2; Examples This function generates <P>

# HTF.TABLECLOSE

This function generates the </TABLE> tag which marks the end of an HTML table.

## Syntax

```sql
HTF.TABLECLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

To define the beginning of an HTML table, use the TABLEOPEN Function . Syntax HTF.TABLECLOSE RETURN VARCHAR2; Examples This function generates </TABLE>

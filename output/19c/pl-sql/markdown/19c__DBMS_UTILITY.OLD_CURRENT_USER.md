# DBMS_UTILITY.OLD_CURRENT_USER

This function returns the session value from sys_context (' userenv ', ' current_user ').

## Syntax

```sql
DBMS_UTILITY.OLD_CURRENT_USER 
 RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

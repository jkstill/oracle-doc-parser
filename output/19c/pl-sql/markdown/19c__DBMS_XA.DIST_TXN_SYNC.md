# DBMS_XA.DIST_TXN_SYNC

This procedure can be used to synchronize in-doubt transactions when one of the Oracle Real Application Clusters (Oracle RAC) instances fails.

## Syntax

```sql
DBMS_XA.DIST_TXN_SYNC;
```

## Usage Notes

Syntax DBMS_XA.DIST_TXN_SYNC;

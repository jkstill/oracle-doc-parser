# DBMS_LOB.FILECLOSEALL

This procedure closes all BFILEs opened in the session.

## Syntax

```sql
DBMS_LOB.FILECLOSEALL;
```

## Usage Notes

Syntax DBMS_LOB.FILECLOSEALL; Exceptions Table 99-35 FILECLOSEALL Procedure Exception Exception Description UNOPENED_FILE No file has been opened in the session.

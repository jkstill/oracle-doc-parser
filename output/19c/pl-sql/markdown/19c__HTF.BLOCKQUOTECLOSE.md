# HTF.BLOCKQUOTECLOSE

This function generates the </BLOCKQUOTE> tag which mark the end of a section of quoted text.

## Syntax

```sql
HTF.BLOCKQUOTECLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You mark the beginning of a section of text by means of the BLOCKQUOTEOPEN Function . Syntax HTF.BLOCKQUOTECLOSE RETURN VARCHAR2; Examples This function generates </BLOCKQUOTE>

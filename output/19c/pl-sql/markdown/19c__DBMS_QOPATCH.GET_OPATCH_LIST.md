# DBMS_QOPATCH.GET_OPATCH_LIST

This function provides list of patches installed as an XML element from the XML inventory.

## Syntax

```sql
DBMS_QOPATCH.GET_OPATCH_LIST
 RETURN XMLTYPE;
```

**Returns:** `XMLTYPE`

## Usage Notes

Syntax DBMS_QOPATCH.GET_OPATCH_LIST RETURN XMLTYPE;

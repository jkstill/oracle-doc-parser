# HTP.PRECLOSE

This procedure generates the </PRE> tag which marks the end of a section of preformatted text in the body of the HTML page.

## Syntax

```sql
HTP.PRECLOSE;
```

## Usage Notes

To mark the beginning of a section of preformatted text in the body of the HTML page, use the PREOPEN Procedure . Examples This procedure generates </PRE>

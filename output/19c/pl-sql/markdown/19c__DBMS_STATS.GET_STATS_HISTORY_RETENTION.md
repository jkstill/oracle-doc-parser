# DBMS_STATS.GET_STATS_HISTORY_RETENTION

This function returns the current statistics history retention value.

## Syntax

```sql
DBMS_STATS.GET_STATS_HISTORY_RETENTION
 RETURN NUMBER;
```

**Returns:** `NUMBER`

## Usage Notes

Syntax DBMS_STATS.GET_STATS_HISTORY_RETENTION RETURN NUMBER; Usage Notes No special privilege or role is needed to invoke this procedure.

# DBMS_TRANSACTION.READ_ONLY

This procedure is equivalent to the SQL statement SET TRANSACTION READ ONLY .

## Syntax

```sql
DBMS_TRANSACTION.READ_ONLY;
```

## Usage Notes

Syntax DBMS_TRANSACTION.READ_ONLY;

# HTF.FORMCLOSE

This function generates the </FORM> tag which marks the end of a form section in an HTML document.

## Syntax

```sql
HTF.FORMCLOSE
  RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

You mark the beginning of the form section by means of the FORMOPEN Function . Syntax HTF.FORMCLOSE RETURN VARCHAR2; Examples This function generates </FORM>

# DBMS_XDB.GETXDB_TABLESPACE

This deprecated function returns the current tablespace of the XDB (user).

## Syntax

```sql
DBMS_XDB.GETXDB_TABLESPACE
 RETURN VARCHAR2;
```

**Returns:** `VARCHAR2`

## Usage Notes

Note: This procedure is deprecated in Release 12 c . This functionality is replaced by a subprogram of the same name in the DBMS_XDB_REPOS package - the GETXDB_TABLESPACE Function .

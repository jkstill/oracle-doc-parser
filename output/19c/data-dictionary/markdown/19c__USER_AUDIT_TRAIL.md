# USER_AUDIT_TRAIL

USER_AUDIT_TRAIL displays the standard audit trail entries related to the current user.

## Usage Notes

Previous Next JavaScript must be enabled to correctly display this content Its columns are the same as those in " DBA_AUDIT_TRAIL " . The view displays audit records generated by actions performed by the user and audit records generated by actions performed on the user’s schema objects. Note: This view is populated only in an Oracle Database where unified auditing is not enabled. When unified auditing is enabled in Oracle Database, the audit records are populated in the new audit trail and can be viewed from UNIFIED_AUDIT_TRAIL . See Oracle Database Security Guide for more information about unified auditing. See Oracle Database Upgrade Guide for more information about migrating to unified auditing.

# USER_CHANGE_NOTIFICATION_REGS

USER_CHANGE_NOTIFICATION_REGS describes the change notification registrations owned by the current user. Its columns (except for USERNAME ) are the same as those in DBA_CHANGE_NOTIFICATION_REGS .

# DBA_APPLY_OBJECT_DEPENDENCIES

DBA_APPLY_OBJECT_DEPENDENCIES displays information about the object dependencies for all apply processes in the database.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| OBJECT_OWNER | VARCHAR2(128) | Owner of the object |
| OBJECT_NAME | VARCHAR2(128) | Name of the object |
| PARENT_OBJECT_OWNER | VARCHAR2(128) | Parent of the object owner |
| PARENT_OBJECT_NAME | VARCHAR2(128) | Parent of the named object |

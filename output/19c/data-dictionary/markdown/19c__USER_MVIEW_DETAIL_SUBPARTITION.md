# USER_MVIEW_DETAIL_SUBPARTITION

USER_MVIEW_DETAIL_SUBPARTITION displays freshness information, with respect to partition change tracking (PCT) detail subpartitions, for the materialized views owned by the current user. Its columns are the same as those in ALL_MVIEW_DETAIL_SUBPARTITION .

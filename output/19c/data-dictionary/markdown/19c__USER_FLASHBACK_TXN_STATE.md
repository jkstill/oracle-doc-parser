# USER_FLASHBACK_TXN_STATE

USER_FLASHBACK_TXN_STATE displays information about the compensating status of the transactions owned by the current user. Its columns (except for USERNAME ) are the same as those in DBA_FLASHBACK_TXN_STATE .

# USER_SCHEDULER_PROGRAM_ARGS

USER_SCHEDULER_PROGRAM_ARGS displays information about the arguments of the Scheduler programs owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_PROGRAM_ARGS .

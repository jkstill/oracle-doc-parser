# RESOURCE_MAP

RESOURCE_MAP describes resources. This table can be used to map resource names to resource numbers.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| RESOURCE# | NUMBER | Numeric resource code |
| TYPE# | NUMBER | Numeric type code |
| NAME | VARCHAR2(32) | Name of the resource |

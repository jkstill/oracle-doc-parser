# V$STATISTICS_LEVEL

V$STATISTICS_LEVEL displays the status of the statistics/advisories controlled by STATISTICS_LEVEL .

| ACTIVATION_LEVEL | VARCHAR2(7) |  |
| STATISTICS_VIEW_NAME | VARCHAR2(64) |  |
| SESSION_SETTABLE | VARCHAR2(3) |  |
| CON_ID | NUMBER |  |

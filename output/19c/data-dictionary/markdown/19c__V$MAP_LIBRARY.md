# V$MAP_LIBRARY

V$MAP_LIBRARY displays a list of all mapping libraries dynamically loaded by the external process.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| LIB_IDX | NUMBER |  |
| LIB_NAME | VARCHAR2(256) |  |
| VENDOR_NAME | VARCHAR2(64) |  |
| PROTOCOL_NUM | NUMBER |  |
| VERSION_NUM | VARCHAR2(32) |  |
| PATH_NAME | VARCHAR2(4000) |  |
| MAP_FILE | VARCHAR2(1) |  |
| FILE_CFGID | VARCHAR2(13) |  |
| MAP_ELEM | VARCHAR2(1) |  |
| ELEM_CFGID | VARCHAR2(13) |  |
| MAP_SYNC | VARCHAR2(1) |  |
| CON_ID | NUMBER |  |

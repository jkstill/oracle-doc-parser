# GLOBAL_NAMES

GLOBAL_NAMES specifies whether a database link is required to have the same name as the database to which it connects.

## Usage Notes

If you use or plan to use distributed processing, then Oracle recommends that you set this parameter to true to ensure the use of consistent naming conventions for databases and links in a networked environment.

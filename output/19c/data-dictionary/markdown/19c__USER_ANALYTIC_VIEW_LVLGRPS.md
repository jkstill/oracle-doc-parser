# USER_ANALYTIC_VIEW_LVLGRPS

USER_ANALYTIC_VIEW_LVLGRPS describes the analytic view measure and level groups of the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_LVLGRPS .

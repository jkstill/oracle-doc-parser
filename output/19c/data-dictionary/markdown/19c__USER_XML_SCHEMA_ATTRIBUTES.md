# USER_XML_SCHEMA_ATTRIBUTES

USER_XML_SCHEMA_ATTRIBUTES describes all the attributes and their properties owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_XML_SCHEMA_ATTRIBUTES .

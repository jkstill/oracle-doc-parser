# USER_CUBE_SUB_PARTITION_LEVELS

USER_CUBE_SUB_PARTITION_LEVELS describes the OLAP secondary partition levels in the database that are owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_CUBE_SUB_PARTITION_LEVELS .

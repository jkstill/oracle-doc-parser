# USER_PARALLEL_EXECUTE_TASKS

USER_PARALLEL_EXECUTE_TASKS displays the tasks created by the current user. Its columns (except for TASK_OWNER ) are the same as those in DBA_PARALLEL_EXECUTE_TASKS .

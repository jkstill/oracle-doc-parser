# DBA_ANALYTIC_VIEW_LEVELS

DBA_ANALYTIC_VIEW_LEVELS describes the levels in the hierarchies in all analytic views in the database. Its columns are the same as those in ALL_ANALYTIC_VIEW_LEVELS .

# USER_ANALYTIC_VIEW_CLASS

USER_ANALYTIC_VIEW_CLASS describes the classifications of the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in USER_ANALYTIC_VIEW_CLASS .

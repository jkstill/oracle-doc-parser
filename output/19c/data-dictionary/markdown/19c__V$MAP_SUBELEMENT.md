# V$MAP_SUBELEMENT

V$MAP_SUBELEMENT displays a list of all subelement mapping structures in the shared memory of the instance.

| Column | Type | Description |
|--------|------|-------------|
| CHILD_IDX | NUMBER |  |
| PARENT_IDX | NUMBER |  |
| SUB_NUM | NUMBER |  |
| SUB_SIZE | NUMBER |  |
| ELEM_OFFSET | NUMBER |  |
| SUB_FLAGS | NUMBER |  |
| CON_ID | NUMBER |  |

# USER_XTERNAL_TAB_PARTITIONS

USER_XTERNAL_TAB_PARTITIONS describes partition-level information for partitioned external tables owned by the current user. Its columns (except for TABLE_OWNER ) are the same as those in ALL_XTERNAL_TAB_PARTITIONS .

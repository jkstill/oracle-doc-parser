# USER_XML_SCHEMA_ELEMENTS

USER_XML_SCHEMA_ELEMENTS describes all the elements and their properties owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_XML_SCHEMA_ELEMENTS .

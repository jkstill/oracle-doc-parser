# USER_XML_SCHEMA_SUBSTGRP_MBRS

USER_XML_SCHEMA_SUBSTGRP_MBRS describes all members of substitution groups owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_XML_SCHEMA_SUBSTGRP_MBRS .

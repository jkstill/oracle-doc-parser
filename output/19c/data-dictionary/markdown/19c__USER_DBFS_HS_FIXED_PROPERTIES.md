# USER_DBFS_HS_FIXED_PROPERTIES

USER_DBFS_HS_FIXED_PROPERTIES shows non-modifiable properties of all Databaase File System (DBFS) hierarchical stores owned by current user. Its columns (except for STORE_OWNER ) are the same as those in DBA_DBFS_HS_FIXED_PROPERTIES .

# USER_JSON_DATAGUIDES

USER_JSON_DATAGUIDES provides information on the JavaScript Object Notation (JSON) columns owned by the current user that have data guide enabled. Its columns (except for OWNER ) are the same as those in ALL_JSON_DATAGUIDES .

# USER_ATTRIBUTE_DIM_KEYS

USER_ATTRIBUTE_DIM_KEYS describes the keys of the attribute dimensions owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_KEYS .

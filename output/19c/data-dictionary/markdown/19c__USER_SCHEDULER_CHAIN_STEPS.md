# USER_SCHEDULER_CHAIN_STEPS

USER_SCHEDULER_CHAIN_STEPS displays information about the defined steps of the chains owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_CHAIN_STEPS .

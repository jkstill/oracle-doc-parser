# USER_ATTRIBUTE_DIM_LEVELS_AE

USER_ATTRIBUTE_DIM_LEVELS_AE describes the levels of the attribute dimensions (across all editions) owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_LEVELS_AE .

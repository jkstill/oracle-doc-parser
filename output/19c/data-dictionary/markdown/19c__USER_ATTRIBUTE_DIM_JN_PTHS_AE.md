# USER_ATTRIBUTE_DIM_JN_PTHS_AE

USER_ATTRIBUTE_DIM_JN_PTHS_AE describes the join paths for the attribute dimensions (across all editions) owned by the current user. Its columns are the same as those in ALL_ATTRIBUTE_DIM_JN_PTHS_AE .

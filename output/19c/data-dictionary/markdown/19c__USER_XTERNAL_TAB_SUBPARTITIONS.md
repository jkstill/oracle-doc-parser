# USER_XTERNAL_TAB_SUBPARTITIONS

USER_XTERNAL_TAB_SUBPARTITIONS describes subpartition-level information for partitioned external tables owned by the current user. Its columns (except for TABLE_OWNER ) are the same as those in ALL_XTERNAL_TAB_SUBPARTITIONS .

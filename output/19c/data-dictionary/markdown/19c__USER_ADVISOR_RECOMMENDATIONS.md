# USER_ADVISOR_RECOMMENDATIONS

USER_ADVISOR_RECOMMENDATIONS displays the results of an analysis of the recommendations owned by the current user. Its columns (except for OWNER ) are the same as those in DBA_ADVISOR_RECOMMENDATIONS .

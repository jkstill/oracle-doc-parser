# DBA_ANALYTIC_VIEW_DIMENSIONS

DBA_ANALYTIC_VIEW_DIMENSIONS describes the attribute dimensions in all analytic views in the database. Its columns are the same as those in ALL_ANALYTIC_VIEW_DIMENSIONS .

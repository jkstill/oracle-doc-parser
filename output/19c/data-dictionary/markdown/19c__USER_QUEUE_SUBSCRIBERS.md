# USER_QUEUE_SUBSCRIBERS

USER_QUEUE_SUBSCRIBERS displays the list of subscribers on queues that are under the current user's schema. Its columns (except for OWNER ) are the same as those in ALL_QUEUE_SUBSCRIBERS .

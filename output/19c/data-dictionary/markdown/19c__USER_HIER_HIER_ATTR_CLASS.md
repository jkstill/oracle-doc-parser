# USER_HIER_HIER_ATTR_CLASS

USER_HIER_HIER_ATTR_CLASS describes the classifications of the hierarchical attributes of the hierarchies owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_HIER_HIER_ATTR_CLASS .

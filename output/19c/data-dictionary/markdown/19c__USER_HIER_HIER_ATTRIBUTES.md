# USER_HIER_HIER_ATTRIBUTES

USER_HIER_HIER_ATTRIBUTES describes the hierarchical attributes of the hierarchies owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_HIER_HIER_ATTRIBUTES .

# USER_ANALYTIC_VIEW_LEVELS

USER_ANALYTIC_VIEW_LEVELS describes the levels in the hierarchies in the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_LEVELS .

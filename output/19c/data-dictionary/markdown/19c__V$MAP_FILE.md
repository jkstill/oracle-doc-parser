# V$MAP_FILE

V$MAP_FILE displays a list of all file mapping structures in the shared memory of the instance.

| Column | Type | Description |

| FILE_MAP_IDX | NUMBER |  |
| FILE_CFGID | VARCHAR2(256) |  |
| FILE_STATUS | VARCHAR2(7) |  |

| FILE_TYPE | VARCHAR2(11) |  |
| FILE_STRUCTURE | VARCHAR2(9) |  |
| FILE_SIZE | NUMBER |  |
| FILE_NEXTS | NUMBER |  |
| LIB_IDX | NUMBER |  |

# DBA_ATTRIBUTE_DIM_LEVELS_AE

DBA_ATTRIBUTE_DIM_LEVELS_AE describes the levels of all attribute dimensions (across all editions) in the database. Its columns are the same as those in ALL_ATTRIBUTE_DIM_LEVELS_AE .

# DBA_ANALYTIC_VIEW_HIERS_AE

DBA_ANALYTIC_VIEW_HIERS_AE describes the hierarchies in all analytic views (across all editions) in the database. Its columns are the same as those in ALL_ANALYTIC_VIEW_HIERS_AE .

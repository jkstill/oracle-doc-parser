# DBA_MVIEW_DETAIL_SUBPARTITION

DBA_MVIEW_DETAIL_SUBPARTITION displays freshness information, with respect to partition change tracking (PCT) detail subpartitions, for all materialized views in the database. Its columns are the same as those in ALL_MVIEW_DETAIL_SUBPARTITION .

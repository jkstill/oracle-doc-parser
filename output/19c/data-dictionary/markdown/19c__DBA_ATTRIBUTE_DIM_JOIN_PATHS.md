# DBA_ATTRIBUTE_DIM_JOIN_PATHS

DBA_ATTRIBUTE_DIM_JOIN_PATHS describes the join paths for all attribute dimensions in the database. Its columns are the same as those in ALL_ATTRIBUTE_DIM_JOIN_PATHS .

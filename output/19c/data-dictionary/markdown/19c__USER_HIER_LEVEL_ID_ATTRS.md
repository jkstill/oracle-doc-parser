# USER_HIER_LEVEL_ID_ATTRS

USER_HIER_LEVEL_ID_ATTRS describes the attributes that uniquely identify members of the levels of the hierarchies owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_HIER_LEVEL_ID_ATTRS .

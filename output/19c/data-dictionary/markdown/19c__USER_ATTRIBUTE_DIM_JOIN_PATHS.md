# USER_ATTRIBUTE_DIM_JOIN_PATHS

USER_ATTRIBUTE_DIM_JOIN_PATHS describes the join paths for the attribute dimensions owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_JOIN_PATHS .

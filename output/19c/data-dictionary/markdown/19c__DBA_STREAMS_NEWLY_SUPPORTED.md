# DBA_STREAMS_NEWLY_SUPPORTED

DBA_STREAMS_NEWLY_SUPPORTED displays information about all tables in the database that are newly supported by capture processes. Its columns are the same as those in ALL_STREAMS_NEWLY_SUPPORTED .

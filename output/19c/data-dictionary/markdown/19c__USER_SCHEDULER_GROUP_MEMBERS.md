# USER_SCHEDULER_GROUP_MEMBERS

USER_SCHEDULER_GROUP_MEMBERS displays information about the members of the Scheduler object groups owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_GROUP_MEMBERS .

# DBA_XTERNAL_TAB_SUBPARTITIONS

DBA_XTERNAL_TAB_SUBPARTITIONS describes subpartition-level information for partitioned external tables in the database. Its columns are the same as those in ALL_XTERNAL_TAB_SUBPARTITIONS .

# USER_MINING_MODEL_ATTRIBUTES

USER_MINING_MODEL_ATTRIBUTES describes the mining model attributes owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_MINING_MODEL_ATTRIBUTES .

# USER_ADVISOR_SQLW_PARAMETERS

USER_ADVISOR_SQLW_PARAMETERS displays the workload parameters and their current values owned by the current user. Its columns (except for OWNER ) are the same as those in DBA_ADVISOR_SQLW_PARAMETERS .

# DBA_MVIEW_DETAIL_PARTITION

DBA_MVIEW_DETAIL_PARTITION displays freshness information, with respect to partition change tracking (PCT) detail partitions, for all materialized views in the database. Its columns are the same as those in ALL_MVIEW_DETAIL_PARTITION .

# USER_XML_SCHEMA_NAMESPACES

USER_XML_SCHEMA_NAMESPACES describes all the available namespaces owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_XML_SCHEMA_NAMESPACES .

# USER_ANALYTIC_VIEW_HIERS

USER_ANALYTIC_VIEW_HIERS describes the hierarchies in the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_HIERS .

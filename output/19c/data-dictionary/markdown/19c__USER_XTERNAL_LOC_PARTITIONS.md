# USER_XTERNAL_LOC_PARTITIONS

USER_XTERNAL_LOC_PARTITIONS describes partition-level locations owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_XTERNAL_LOC_PARTITIONS .

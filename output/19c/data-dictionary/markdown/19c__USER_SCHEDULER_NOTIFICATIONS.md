# USER_SCHEDULER_NOTIFICATIONS

USER_SCHEDULER_NOTIFICATIONS displays information about the E-mail notifications for the jobs owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_NOTIFICATIONS .

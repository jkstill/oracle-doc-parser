# DBA_ATTRIBUTE_DIM_ORDER_ATTRS

DBA_ATTRIBUTE_DIM_ORDER_ATTRS describes the order attributes of all attribute dimensions in the database. Its columns are the same as those in ALL_ATTRIBUTE_DIM_ORDER_ATTRS .

# RESOURCE_COST

RESOURCE_COST lists the cost for each resource.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| RESOURCE_NAME | VARCHAR2(32) | Name of the resource |
| UNIT_COST | NUMBER | Cost of the resource |

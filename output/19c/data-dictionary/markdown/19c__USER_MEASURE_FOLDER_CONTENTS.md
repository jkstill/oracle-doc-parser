# USER_MEASURE_FOLDER_CONTENTS

USER_MEASURE_FOLDER_CONTENTS describes the contents of the OLAP measure folders owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_MEASURE_FOLDER_CONTENTS .

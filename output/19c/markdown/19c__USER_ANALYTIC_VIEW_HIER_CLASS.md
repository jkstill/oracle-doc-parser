# USER_ANALYTIC_VIEW_HIER_CLASS

USER_ANALYTIC_VIEW_HIER_CLASS describes the classifications of the hierarchies in the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_HIER_CLASS .

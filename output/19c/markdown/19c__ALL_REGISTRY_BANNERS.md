# ALL_REGISTRY_BANNERS

ALL_REGISTRY_BANNERS displays the valid components loaded into the database.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| BANNER | VARCHAR2(80) | Component display banner |
| BANNER_FULL | VARCHAR2(80) | Component display banner with full version |

## Usage Notes

Column Datatype NULL Description BANNER VARCHAR2(80) Component display banner BANNER_FULL VARCHAR2(80) Component display banner with full version

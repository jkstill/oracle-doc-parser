# USER_BLOCKCHAIN_TABLES

USER_BLOCKCHAIN_TABLES describes the blockchain tables owned by the current user. Its columns (except for SCHEMA_NAME ) are the same as those in ALL_BLOCKCHAIN_TABLES .

## Usage Notes

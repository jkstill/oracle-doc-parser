# NLS_NCHAR_CONV_EXCP

NLS_NCHAR_CONV_EXCP determines whether an error is reported when there is data loss during an implicit or explicit character type conversion between NCHAR/NVARCHAR2 and CHAR/VARCHAR2.

## Usage Notes

The default value results in no error being reported. See Also: Oracle Database Globalization Support Guide for more information about this parameter See Also: Oracle Database Globalization Support Guide for more information about this parameter

# USER_CUBE_DIMENSIONALITY

USER_CUBE_DIMENSIONALITY describes the dimension order for the OLAP cubes owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_CUBE_DIMENSIONALITY .

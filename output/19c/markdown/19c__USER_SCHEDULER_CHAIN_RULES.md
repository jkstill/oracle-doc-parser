# USER_SCHEDULER_CHAIN_RULES

USER_SCHEDULER_CHAIN_RULES displays information about the rules for the chains owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_CHAIN_RULES .

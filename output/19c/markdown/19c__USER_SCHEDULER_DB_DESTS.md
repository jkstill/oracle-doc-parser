# USER_SCHEDULER_DB_DESTS

USER_SCHEDULER_DB_DESTS displays information about the destination objects owned by the current user pointing to remote databases. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_DB_DESTS .

# USER_ANALYTIC_VIEW_KEYS

USER_ANALYTIC_VIEW_KEYS describes the key columns of the attribute dimensions in the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_KEYS .

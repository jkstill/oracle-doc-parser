# V$ASM_USERGROUP_MEMBER

V$ASM_USERGROUP_MEMBER displays the members for each Oracle Automatic Storage Management (Oracle ASM) File Access Control group.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| GROUP_NUMBER | NUMBER |  |
| MEMBER_NUMBER | NUMBER |  |
| MEMBER_INCARNATION | NUMBER |  |
| USERGROUP_NUMBER | NUMBER |  |
| USERGROUP_INCARNATION | NUMBER |  |
| CON_ID | NUMBER |  |

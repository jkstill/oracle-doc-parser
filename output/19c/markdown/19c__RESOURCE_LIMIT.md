# RESOURCE_LIMIT

RESOURCE_LIMIT determines whether resource limits are enforced in database profiles.

## Usage Notes

TRUE Enables the enforcement of resource limits FALSE Disables the enforcement of resource limits See Also: Oracle Database Administrator’s Guide and Oracle Database SQL Language Reference for more information on setting resource limits for profiles See Also: Oracle Database Administrator’s Guide and Oracle Database SQL Language Reference for more information on setting resource limits for profiles

# USER_SQL_TRANSLATION_PROFILES

USER_SQL_TRANSLATION_PROFILES describes all SQL translation profiles owned by the user. Its columns (except for OWNER ) are the same as those in ALL_SQL_TRANSLATION_PROFILES .

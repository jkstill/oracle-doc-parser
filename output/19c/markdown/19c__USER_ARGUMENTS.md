# USER_ARGUMENTS

USER_ARGUMENTS lists the arguments of the functions and procedures that are owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ARGUMENTS .

## Usage Notes

See Also: " ALL_ARGUMENTS " " USER_PROCEDURES " for information about the functions and procedures that are owned by the current user See Also: " ALL_ARGUMENTS " " USER_PROCEDURES " for information about the functions and procedures that are owned by the current user

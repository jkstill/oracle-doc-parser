# TABLE_PRIVILEGE_MAP

TABLE_PRIVILEGE_MAP describes privilege (auditing option) type codes.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| PRIVILEGE | NUMBER | Numeric privilege (auditing option) type code |
| NAME | VARCHAR2(40) | Name of the type of privilege (auditing option) |

## Usage Notes

This table can be used to map privilege (auditing option) type numbers to type names.

# USER_XTERNAL_LOC_SUBPARTITIONS

USER_XTERNAL_LOC_SUBPARTITIONS describes subpartition-level locations owned by the current user. Its columns (except for TABLE_OWNER ) are the same as those in ALL_XTERNAL_LOC_SUBPARTITIONS .

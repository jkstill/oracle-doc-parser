# USER_MVIEW_DETAIL_PARTITION

USER_MVIEW_DETAIL_PARTITION displays freshness information, with respect to partition change tracking (PCT) detail partitions, for the materialized views owned by the current user. Its columns are the same as those in ALL_MVIEW_DETAIL_PARTITION .

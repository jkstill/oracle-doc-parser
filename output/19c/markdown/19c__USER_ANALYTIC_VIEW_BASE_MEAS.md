# USER_ANALYTIC_VIEW_BASE_MEAS

USER_ANALYTIC_VIEW_BASE_MEAS describes the base measures in the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_BASE_MEAS .

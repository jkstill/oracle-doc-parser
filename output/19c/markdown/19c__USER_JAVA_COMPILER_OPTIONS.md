# USER_JAVA_COMPILER_OPTIONS

USER_JAVA_COMPILER_OPTIONS displays information about the native compiler options owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_JAVA_COMPILER_OPTIONS .

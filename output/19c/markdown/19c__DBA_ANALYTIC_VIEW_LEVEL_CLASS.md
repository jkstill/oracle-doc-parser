# DBA_ANALYTIC_VIEW_LEVEL_CLASS

DBA_ANALYTIC_VIEW_LEVEL_CLASS describes the level classifications of all analytic views in the database. Its columns are the same as those in ALL_ANALYTIC_VIEW_LEVEL_CLASS .

# USER_XML_SCHEMA_SIMPLE_TYPES

USER_XML_SCHEMA_SIMPLE_TYPES describes all simple types owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_XML_SCHEMA_SIMPLE_TYPES .

# USER_ATTRIBUTE_DIM_TABLES_AE

USER_ATTRIBUTE_DIM_TABLES_AE describes the tables used by the attribute dimensions (across all editions) owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_TABLES_AE .

## Usage Notes

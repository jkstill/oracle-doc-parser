# USER_STORED_SETTINGS

USER_STORED_SETTINGS lists information about the persistent parameter settings for stored PL/SQL units, but only shows information about PL/SQL units owned by the current user.

## Usage Notes

USER_STORED_SETTINGS does not display the OWNER column. The rest of its columns are the same as those in ALL_STORED_SETTINGS . See Also: " ALL_STORED_SETTINGS " See Also: " ALL_STORED_SETTINGS "

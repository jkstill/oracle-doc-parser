# NLS_DATABASE_PARAMETERS

NLS_DATABASE_PARAMETERS lists permanent NLS parameters of the database.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| PARAMETER | VARCHAR2(128) | Parameter name |
| VALUE | VARCHAR2(64) | Parameter value |

# USER_ATTRIBUTE_TRANSFORMATIONS

USER_ATTRIBUTE_TRANSFORMATIONS describes the transformation functions for the transformations owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_TRANSFORMATIONS .

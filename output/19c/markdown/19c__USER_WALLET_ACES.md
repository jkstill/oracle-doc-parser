# USER_WALLET_ACES

USER_WALLET_ACES describes the status of access control entries for the current user to access wallets through PL/SQL network utility packages.

## Usage Notes

Its columns (except for ACE_ORDER , START_DATE , END_DATE , GRANT_TYPE , INVERTED_PRINCIPAL , PRINCIPAL , PRINCIPAL_TYPE, and STATUS ) are the same as those in DBA_WALLET_ACES . See Also: " DBA_WALLET_ACES " See Also: " DBA_WALLET_ACES "

# NLS_SESSION_PARAMETERS

NLS_SESSION_PARAMETERS lists NLS parameters of the user session.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| PARAMETER | VARCHAR2(30) | Parameter name |
| VALUE | VARCHAR2(64) | Parameter value |

# NLS_INSTANCE_PARAMETERS

NLS_INSTANCE_PARAMETERS lists NLS parameters of the instance.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| PARAMETER | VARCHAR2(30) | Parameter name |
| VALUE | VARCHAR2(64) | Parameter value |

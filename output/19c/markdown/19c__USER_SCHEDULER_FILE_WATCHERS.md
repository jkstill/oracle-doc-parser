# USER_SCHEDULER_FILE_WATCHERS

USER_SCHEDULER_FILE_WATCHERS displays information about the Scheduler file watch requests owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_FILE_WATCHERS .

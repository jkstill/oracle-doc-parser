# V$IOFUNCMETRIC_HISTORY

V$IOFUNCMETRIC_HISTORY displays a recent history of the I/O statistics information by database function for the most recent time interval period.

## Usage Notes

The columns for V$IOFUNCMETRIC_HISTORY are the same as those for V$IOFUNCMETRIC . See Also: " V$IOFUNCMETRIC " See Also: " V$IOFUNCMETRIC "

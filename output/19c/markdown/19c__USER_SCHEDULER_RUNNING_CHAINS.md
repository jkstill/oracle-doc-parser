# USER_SCHEDULER_RUNNING_CHAINS

USER_SCHEDULER_RUNNING_CHAINS displays information about the chain steps of the running chains owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_RUNNING_CHAINS .

# USER_ADVISOR_SQLW_TABLES

USER_ADVISOR_SQLW_TABLES displays cross references between the workload statements and the tables referenced in the statement. Its columns (except for OWNER ) are the same as those in DBA_ADVISOR_SQLW_TABLES .

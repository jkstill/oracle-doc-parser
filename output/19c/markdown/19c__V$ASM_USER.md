# V$ASM_USER

V$ASM_USER displays the effective operating system user names of connected database instances and of file owners.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| GROUP_NUMBER | NUMBER |  |
| USER_NUMBER | NUMBER |  |
| COMPOUND_INDEX | NUMBER |  |
| INCARNATION | NUMBER |  |
| CLUSTER_ID | VARCHAR2(128) |  |
| OS_ID | VARCHAR2(128) |  |
| OS_NAME | VARCHAR2(33) |  |
| CON_ID | NUMBER |  |

# DBA_ANALYTIC_VIEW_KEYS

DBA_ANALYTIC_VIEW_KEYS describes the key columns of the attribute dimensions in all analytic views in the database. Its columns are the same as those in ALL_ANALYTIC_VIEW_KEYS .

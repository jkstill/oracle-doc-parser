# USER_JAVA_DERIVATIONS

USER_JAVA_DERIVATIONS displays mapping information about Java source objects and their derived Java class objects and Java resource objects for the Java classes owned by the current user.

## Usage Notes

Its columns (except for OWNER ) are the same as those in ALL_JAVA_DERIVATIONS . See Also: " ALL_JAVA_DERIVATIONS " See Also: " ALL_JAVA_DERIVATIONS "

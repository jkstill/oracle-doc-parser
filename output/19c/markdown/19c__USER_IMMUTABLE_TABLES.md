# USER_IMMUTABLE_TABLES

USER_IMMUTABLE_TABLES describes the immutable tables owned by the current user. Its columns (except for SCHEMA_NAME ) are the same as those in ALL_IMMUTABLE_TABLES .

## Usage Notes

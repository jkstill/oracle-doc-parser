# USER_TAB_PARTITIONS

USER_TAB_PARTITIONS describes partition-level partitioning information, partition storage parameters, and partition statistics generated by the DBMS_STATS package for all partitions owned by the current user.

## Usage Notes

Its columns are the same as those in ALL_TAB_PARTITIONS . See Also: " ALL_TAB_PARTITIONS " See Also: " ALL_TAB_PARTITIONS "

# USER_ADVISOR_SQLA_WK_MAP

USER_ADVISOR_SQLA_WK_MAP displays the workload references for the tasks owned by the current user. Its columns (except for OWNER ) are the same as those in DBA_ADVISOR_SQLA_WK_MAP .

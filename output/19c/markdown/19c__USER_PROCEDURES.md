# USER_PROCEDURES

USER_PROCEDURES lists all functions and procedures that are owned by the current user, along with their associated properties. Its columns (except OWNER ) are the same as those in ALL_PROCEDURES .

## Usage Notes

See Also: " ALL_PROCEDURES " " USER_ARGUMENTS " for information about the arguments of the functions and procedures owned by the current user See Also: " ALL_PROCEDURES " " USER_ARGUMENTS " for information about the arguments of the functions and procedures owned by the current user

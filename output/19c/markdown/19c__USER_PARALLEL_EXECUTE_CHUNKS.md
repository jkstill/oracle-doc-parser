# USER_PARALLEL_EXECUTE_CHUNKS

USER_PARALLEL_EXECUTE_CHUNKS displays the chunks for tasks created by the current user. Its columns (except for TASK_OWNER ) are the same as those in DBA_PARALLEL_EXECUTE_CHUNKS .

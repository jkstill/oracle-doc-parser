# USER_HIER_COLUMNS_AE

USER_HIER_COLUMNS_AE describes the columns of the hierarchies (across all editions) owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_HIER_COLUMNS_AE .

## Usage Notes

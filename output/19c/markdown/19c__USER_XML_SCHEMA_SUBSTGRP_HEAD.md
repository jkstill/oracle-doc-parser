# USER_XML_SCHEMA_SUBSTGRP_HEAD

USER_XML_SCHEMA_SUBSTGRP_HEAD describes the heads of substitution groups owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_XML_SCHEMA_SUBSTGRP_HEAD .

# USER_ANALYTIC_VIEW_KEYS_AE

USER_ANALYTIC_VIEW_KEYS_AE describes the key columns of the attribute dimensions in the analytic views (across all editions) owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_KEYS_AE .

## Usage Notes

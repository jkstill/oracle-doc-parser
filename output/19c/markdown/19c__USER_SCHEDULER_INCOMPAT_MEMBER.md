# USER_SCHEDULER_INCOMPAT_MEMBER

USER_SCHEDULER_INCOMPAT_MEMBER displays all Scheduler incompatibility resource objects members owned by the current user. Its columns are the same as those in ALL_SCHEDULER_INCOMPAT_MEMBER .

# USER_EXTERNAL_LOCATIONS

USER_EXTERNAL_LOCATIONS describes the locations (data sources) of the external tables owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_EXTERNAL_LOCATIONS .

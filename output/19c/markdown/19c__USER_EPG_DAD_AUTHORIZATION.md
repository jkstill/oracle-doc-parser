# USER_EPG_DAD_AUTHORIZATION

USER_EPG_DAD_AUTHORIZATION describes the DADs that are authorized to use the user's privileges. Its columns (except for USERNAME ) are the same as those in DBA_EPG_DAD_AUTHORIZATION .

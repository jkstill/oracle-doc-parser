# USER_RSRC_CONSUMER_GROUP_PRIVS

USER_RSRC_CONSUMER_GROUP_PRIVS displays information about the resource consumer groups to which the current user is assigned. Its columns (except for GRANTEE ) are the same as those in DBA_RSRC_CONSUMER_GROUP_PRIVS .

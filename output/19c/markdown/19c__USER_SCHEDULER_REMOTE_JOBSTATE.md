# USER_SCHEDULER_REMOTE_JOBSTATE

USER_SCHEDULER_REMOTE_JOBSTATE displays information about the state of the jobs owned by the current user at remote databases. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_REMOTE_JOBSTATE .

# USER_ADVISOR_EXEC_PARAMETERS

USER_ADVISOR_EXEC_PARAMETERS displays the parameter values used for past executions of tasks owned by the current user. Its columns (except for OWNER ) are the same as those in DBA_ADVISOR_EXEC_PARAMETERS .

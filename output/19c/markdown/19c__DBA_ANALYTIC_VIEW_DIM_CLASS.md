# DBA_ANALYTIC_VIEW_DIM_CLASS

DBA_ANALYTIC_VIEW_DIM_CLASS describes the classifications of the attribute dimensions in all analytic views in the database. Its columns are the same as those in ALL_ANALYTIC_VIEW_DIM_CLASS .

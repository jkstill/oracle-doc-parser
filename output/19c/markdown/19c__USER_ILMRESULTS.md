# USER_ILMRESULTS

USER_ILMRESULTS displays information on data movement-related Automatic Data Optimization jobs for tasks created by the user.

## Usage Notes

Its columns are the same as those in DBA_ILMRESULTS . Note: Automatic Data Optimization is supported in Oracle Database 12 c Release 2 multitenant environments.

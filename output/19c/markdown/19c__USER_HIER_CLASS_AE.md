# USER_HIER_CLASS_AE

USER_HIER_CLASS_AE describes the classifications of the hierarchies (across all editions) owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_HIER_CLASS_AE .

## Usage Notes

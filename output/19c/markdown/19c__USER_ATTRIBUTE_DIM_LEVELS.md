# USER_ATTRIBUTE_DIM_LEVELS

USER_ATTRIBUTE_DIM_LEVELS describes the levels of the attribute dimensions owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_LEVELS .

# USER_ATTRIBUTE_DIM_ATTRS

USER_ATTRIBUTE_DIM_ATTRS describes the attributes of the attribute dimensions owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_ATTRS .

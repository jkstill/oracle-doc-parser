# DBA_SCHEDULER_RSC_CONSTRAINTS

DBA_SCHEDULER_RSC_CONSTRAINTS lists all Oracle Scheduler resource constraint members in the database. Its columns are the same as those in ALL_SCHEDULER_RSC_CONSTRAINTS .

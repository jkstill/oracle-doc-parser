# USER_SCHEDULER_RUNNING_JOBS

USER_SCHEDULER_RUNNING_JOBS displays information about the running Scheduler jobs owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_RUNNING_JOBS .

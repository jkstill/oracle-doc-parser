# DBA_TAB_PARTITIONS

DBA_TAB_PARTITIONS displays partition-level partitioning information, partition storage parameters, and partition statistics generated by the DBMS_STATS package for all partitions in the database.

## Usage Notes

Its columns are the same as those in " ALL_TAB_PARTITIONS " .

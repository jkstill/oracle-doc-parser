# USER_ATTRIBUTE_DIM_LVL_CLASS

USER_ATTRIBUTE_DIM_LVL_CLASS describes the level classifications of the attribute dimensions owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_LVL_CLASS .

# USER_SCHEDULER_RSC_CONSTRAINTS

USER_SCHEDULER_RSC_CONSTRAINTS lists all Oracle Scheduler resource constraint members owned by the current user. Its columns are the same as those in ALL_SCHEDULER_RSC_CONSTRAINTS .

# DBA_APPLY_ERROR_MESSAGES

DBA_APPLY_ERROR_MESSAGES displays information about the individual messages in all of the error transactions generated by all apply processes in the database. Its columns are the same as those in ALL_APPLY_ERROR_MESSAGES .

## Usage Notes

For XStream inbound servers, each message in an error transaction is an LCR. Note: Messages that were spilled from memory to hard disk do not appear in this view. This view does not contain information related to XStream outbound servers.

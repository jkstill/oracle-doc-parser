# USER_ANALYTIC_VIEW_ATTR_CLASS

USER_ANALYTIC_VIEW_ATTR_CLASS describes the attribute classifications of the analytic views owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ANALYTIC_VIEW_ATTR_CLASS .

# USER_ATTRIBUTE_DIMENSIONS

USER_ATTRIBUTE_DIMENSIONS describes the attribute dimensions owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIMENSIONS .

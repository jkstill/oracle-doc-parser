# USER_COMPARISON_SCAN_VALUES

USER_COMPARISON_SCAN_VALUES displays information about the values for the comparison scans owned by the current user. Its columns (except for OWNER ) are the same as those in DBA_COMPARISON_SCAN_VALUES .

# DBA_HIERARCHIES_AE

DBA_HIERARCHIES_AE describes all hierarchies (across all editions) in the database. Its columns are the same as those in ALL_HIERARCHIES_AE .

## Usage Notes

Note: This view is available starting with Oracle Database release 19c, version 19.13. See Also: " ALL_HIERARCHIES_AE " Note: This view is available starting with Oracle Database release 19c, version 19.13. See Also: " ALL_HIERARCHIES_AE "

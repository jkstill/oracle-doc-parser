# USER_SCHEDULER_INCOMPATS

USER_SCHEDULER_INCOMPATS displays all Scheduler incompatibility resource objects owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_SCHEDULER_INCOMPATS .

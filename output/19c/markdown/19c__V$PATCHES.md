# V$PATCHES

V$PATCHES shows the patches applied on an Oracle ASM instance and the list of patches applied to an Oracle Grid infrastructure home directory.

## Columns

| Column | Type | Description |
|--------|------|-------------|
| PATCH_ID | NUMBER |  |
| CON_ID | NUMBER |  |

## Usage Notes

Note: If this view is queried in an RDBMS instance, no patch information is returned. Note: If this view is queried in an RDBMS instance, no patch information is returned.

# DBA_TAB_SUBPARTITIONS

DBA_TAB_SUBPARTITIONS displays, for each table subpartition, the subpartition name, name of the table and partition to which it belongs, its storage attributes, and statistics generated by the DBMS_STATS package.

## Usage Notes

Its columns are the same as those in " ALL_TAB_SUBPARTITIONS " .

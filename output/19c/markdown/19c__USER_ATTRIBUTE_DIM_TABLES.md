# USER_ATTRIBUTE_DIM_TABLES

USER_ATTRIBUTE_DIM_TABLES describes the tables used by the attribute dimensions owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_ATTRIBUTE_DIM_TABLES .

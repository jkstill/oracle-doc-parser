# USER_CLUSTERING_JOINS

USER_CLUSTERING_JOINS describes joins to the dimension tables associated with tables with an attribute clustering clause owned by the user. Its columns (except for OWNER ) are the same as those in ALL_CLUSTERING_JOINS .

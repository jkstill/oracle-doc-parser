# DBA_ATTRIBUTE_DIM_KEYS_AE

DBA_ATTRIBUTE_DIM_KEYS_AE describes the keys of all attribute dimensions (across all editions) in the database. Its columns are the same as those in ALL_ATTRIBUTE_DIM_KEYS_AE .

## Usage Notes

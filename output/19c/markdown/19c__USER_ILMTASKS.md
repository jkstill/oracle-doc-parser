# USER_ILMTASKS

USER_ILMTASKS displays information on Automatic Data Optimization tasks created by a user. Its columns are the same as those in DBA_ILMTASKS .

## Usage Notes

Note: Automatic Data Optimization is supported in Oracle Database 12 c Release 2 multitenant environments. See Also: " DBA_ILMTASKS " Note: Automatic Data Optimization is supported in Oracle Database 12 c Release 2 multitenant environments. See Also: " DBA_ILMTASKS "

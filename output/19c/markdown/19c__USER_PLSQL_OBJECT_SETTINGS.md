# USER_PLSQL_OBJECT_SETTINGS

USER_PLSQL_OBJECT_SETTINGS displays information about the compiler settings for the stored objects owned by the current user. Its columns (except for OWNER ) are the same as those in ALL_PLSQL_OBJECT_SETTINGS .

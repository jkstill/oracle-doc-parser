# USER_HIVE_PART_KEY_COLUMNS

USER_HIVE_PART_KEY_COLUMNS provides information about all Hive table partition columns owned by the current user in the database. Its columns are the same as those in ALL_HIVE_PART_KEY_COLUMNS .
